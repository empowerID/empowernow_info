Agent‑Safe Tools

Let agents use tools without fear. The MCP Gateway + ARIA Shield enforce schema pins, plan contracts, parameter allowlists, and egress controls—then prove it with signed receipts.

Controls at a glance

  • Schema pins (version/hash) with safe rollout window
  • Plan contracts — bind steps, params, and per‑step limits
  • Egress allowlists + redirect re‑checks
  • Param allowlists and server‑side data scopes
  • Signed, hash‑chained receipts for every call

Outcomes

Stop breakage

Schema pins catch tool drift before agents hit runtime errors.

Reduce exfiltration

Egress allowlists and redirect re‑checks cut off risky destinations.

Prove compliance

Receipts make decisions and actions tamper‑evident for audits.

How it works

  1. IdP issues an ARIA Passport with pairwise binding, schema pins, and optional plan contract.
  2. Gateway verifies pins, enforces the current plan step, and evaluates PDP for policy.
  3. Request is shaped by parameter allowlists and data scopes; egress is pinned.
  4. On success, a signed receipt records the policy snapshot, hashes, and lineage.
