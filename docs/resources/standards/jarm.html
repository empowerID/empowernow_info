What is JARM?

JWT Secured Authorization Response Mode wraps authorization response parameters in a JWS/JWE for integrity and confidentiality.

Why it matters

  • Defends against parameter tampering and leakage
  • Required by high‑assurance profiles like FAPI 2.0
