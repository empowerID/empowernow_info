What is PAR (Pushed Authorization Requests)?

PAR allows a client to push the authorization request to the Authorization Server, which returns a request URI for the front‑channel authorization step.

Why it matters

  • Keeps sensitive parameters off the browser URL
  • Supports complex objects like RAR authorization details
  • Improves integrity and confidentiality of requests
