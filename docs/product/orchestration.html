Orchestration — Self‑Driving Workflows

No‑code agent tools and workflows. Zero‑shot responses guide agents with next steps, decisions, and AI‑native visualizations—under policy.

Key capabilities

  • Zero‑shot detection and responses tailored for agents
  • Global Next Paths and node‑centric decisions
  • AI‑native Mermaid diagrams (full and per‑node)
  • Enhanced response: ai_context, next_paths, node_specific_next_paths, node_mermaid_diagrams, domain_insights
  • Resumable execution: /start and /resume/{task_id}
