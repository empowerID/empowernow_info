PDP Evaluation API

Endpoints

POST /access/v1/evaluation
POST /access/v1/evaluations  (batch)
GET  /api/v1/applications
GET  /api/v1/applications/{app_id}
GET  /api/v1/applications/{app_id}/policies
POST /api/v1/applications/{app_id}/evaluate
GET  /api/v1/debug/applications/{app_id}/policy-resolution

Request (application‑only)

{
  "subject": {"id": "user123"},
  "resource": {"type": "document", "id": "doc-123", "properties": {"pdp_application": "sharepoint-prod"}},
  "action": {"name": "read"}
}

Response

{
  "allowed": true,
  "obligations": [{"id": "log_access", "attributes": {"level": "detailed"}}],
  "constraints": []
}

Error codes: 400, 404, 409, 500.
